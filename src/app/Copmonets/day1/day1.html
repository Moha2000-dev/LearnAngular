Day 1 - Introduction, Setup & Tailwind CSS

What is Angular?

Angular is a platform and framework for building client-side applications using HTML, CSS, and TypeScript. It is maintained by Google and allows developers to create dynamic and modern web applications.

Tools Required

  • Node.js and npm
  • Angular CLI
  • Code editor (e.g., VS Code)
  • Web browser (Chrome, Firefox, etc.)

What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework for rapidly building custom designs. It provides low-level utility classes that make it easy to style elements directly in your HTML.
